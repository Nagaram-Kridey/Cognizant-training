# Java JUnit Setup

This project demonstrates how to set up and run JUnit tests in a Maven-based Java project.

## JUnit Version
- JUnit 4.13.2

## Project Structure
- `Calculator.java`: A simple class with an `add` method.
- `CalculatorTest.java`: A JUnit test case to test the `add` method.
